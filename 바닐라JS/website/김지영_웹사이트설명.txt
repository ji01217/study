처음 웹사이트를 들어가면 닉네임을 입력하라는 창이 뜨고, 닉네임을 입력하고 로그인 버튼을 누르면 닉네임을 로컬스토리지에 저장한다.
로그인을 하면 로컬스토리지에 저장된 닉네임을 불러와 환영하는 문구와 함께 크게 4가지 기능을 이용할 수 있다.
1. 자바스크립트에 내장된 함수를 이용하여 현재 날짜와 요일, 현재 시간을 초단위로 갱신하여 보여주는 기능
2. 자바스크립트에 내장된 함수를 이용하여 사용자 위치의 위도와 경도를 읽어 api를 이용하여 사용자가 현재 위치하고 있는 도시와 그곳의 날씨를 보여주는 기능
3. Todo list를 추가하여 로컬스토리지에 저장하고, 삭제하는 기능
4. 랜덤으로 노래를 추천해주는 기능
마지막으로 로그아웃을 하면 로컬스토리지에 저장되어있는 정보(닉네임, Todo list)가 모두 삭제되고 처음의 화면으로 돌아간다.

추가하고 싶은 기능
- 노래 api를 받아와서 장르나 사용자의 국가에 따라 노래를 추천하는 기능
- 로그아웃을 할 때 모든 정보가 삭제되지 않고 로컬스토리지에 닉네임과 Todo list를 함께 저장하여 로그아웃 했다가 그 닉네임으로 로그인하면 이전의 Todo list가 다시 보이도록 하는 기능
